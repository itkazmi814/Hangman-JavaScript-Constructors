# Hangman: JavaScript Constructors, Node.js

## Description

Play hangman game made in Node focusing on JavaScript constructors.

## Functionality

Run the application in Node.js with the command ``` node cli.js ```

Enter new letters and keep guessing until you run out of lives.

## Tools & Technologies

* JavaScript, Node
